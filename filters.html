Filters Used in Solana Mempool Sniper

  • Blacklist Filter:
    Skips pools if either token is in the blacklist.
    if BLACKLIST.contains(token_a) || BLACKLIST.contains(token_b)
  • Whitelist Filter:
    If whitelist is not empty, only allows pools where at least one token is in the whitelist.
    if !WHITELIST.is_empty() && !WHITELIST.contains(token_a) && !WHITELIST.contains(token_b)
  • Price Filter:
    Skips pools if the pool price is below the minimum threshold.
    if pool_price < min_price
  • Volume Filter:
    Skips pools if the pool volume is below the minimum threshold.
    if pool_volume < min_vol
  • Already Sniped Filter:
    Skips pools that have already been sniped.
    if RECENT_SNIPES.contains(pool_id)
  • Concurrency Filter:
    Ensures only one snipe is in progress at a time.
    SNIPE_SEMAPHORE.try_acquire()
  • Liquidity Filter:
    Skips pools if both token liquidities are below the minimum.
    if coin_liq < MIN_SOL_LIQUIDITY && pc_liq < MIN_SOL_LIQUIDITY
  • Simulation Mode Filter:
    If simulation mode is enabled, does not execute real snipes.
    if simulate
